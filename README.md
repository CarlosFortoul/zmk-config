
# Corne ZMK Layout

This repository contains my personal Corne layout for ZMK, inspired by the Miryoku philosophy and adapted to my own preferences.

Reference inspiration: [Miryoku README](https://github.com/manna-harbour/miryoku/blob/master/docs/reference/readme.org)
